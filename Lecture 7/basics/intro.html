nomenclature

- rules for variable or bucket declaration
- numbers are allowed
- underscores are allowed
- but you can not start the name of the bucket with a number
- special characters are not allowed.
- keywords can not be used as bucket names.

String template literal

>> let fname = "Aashita"
<- undefined
>> let sname = "Rajput"
<- undefined
>> let intro = `My full, name is ${fname} ${sname} and i am learning`
<- undefined
>> intro
<- "My full, name is Aashita Rajput and i am learning"
